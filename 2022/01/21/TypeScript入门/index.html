TypeScript 的优势

  • 支持 ES6 规范
  • 强大的 IDE 支持
  • Angular2 的开发语言

  • 使用在线 compiler 开发

  • 使用本地 compiler 开发

Typescript 特性

字符串特性

  • 多行字符串 用``声明字符串

  • 字符串模板

  • 自动拆分字符串

参数特性

  • 参数类型：在参数名称后面使用冒号来指定参数的类型
  • 默认参数：在参数声明后面用等号来指定参数的默认值
  • 可选参数：在方法的参数声明后面用问号来标明此参数为可选参数（可选参数必须声明在必选参数后面）

函数特性

  • Rest and Spread 操作符（…）： 用来声明任意数量的方法参数
  • generator 函数：控制函数的执行过程，手动暂定和恢复代码执行
  • destructuring 析构表达式：通过表达式将对象或数组拆解成任意数量的变量

表达式和循环

  • 箭头表达式：用来声明匿名函数，消除传统匿名函数的的 this 指针问题
  • forEach(), for in 和 for of

面向对象特性

  • 类（class）
    • 类是 TypeScript 的核心，使用 TypeScript 开发时，大部分代码都是写在类里面的。这里会介绍类的定义，构造函数，以及类的继承
  • 泛型（generic）
    • 参数化的类型，一般用来限制集合的内容
  • 接口（Interface）
    • 用来建立某种代码约定，使得其他开发者在调用某个方法或创建新的类时必须遵循接口所定义的代码约定
  • 模块(Module)
    • 模块可以帮助开发者讲代码分割为可重用的单元，开发者可以自己决定模块中的哪些资源（类、方法、变量）暴露出去供外部使用，哪些资源只在模块内部使用。 // 一个文件就是一个模块，用 export、import（导出、导入）两个关键字来支撑模块的特性

  • 注解（annotation）

    • 注解为程序的元素（类、方法、变量）加上更直观更明了的说明，这些说明信息与程序的业务逻辑无关，而是供指定的工具或框架来使用的

  • 类型定义文件（*.d.ts）

    • 类型定义文件用来帮助开发者在 TypeScript 中使用已有的 JavaScript 的工具包。如：JQuery
